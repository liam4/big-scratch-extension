Logic Sub-extension

The Logic sub-extension comes with blocks that help with logic gates. It also has two convinient true/false blocks.

<true>
Returns true.
<false>
Returns false.

<() nand ()> (not and)
Returns true if either is false; return false otherwise.
<() nor ()> (not or)
Returns true if neither is true; return false otherwise.
<() xor ()> (exclusive or)
Returns true if exactly one of the two inputs are true; return false otherwise.
<() xnor ()> (exclusive not or)
Returns false if exactly one of the two inputs are true; return true otherwise.

See also:

  • http://www.ee.surrey.ac.uk/Projects/CAL/digital-logic/gatesfunc/index.html, for more information on logic gates, as well as some truth tables to help (note that XOR is the same as EXOR).
