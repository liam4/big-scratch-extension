Math Sub-extension

The Math sub-extension has blocks that handle more operations, and has some constant blocks containing values for various contents that are commonly used in mathematics.

<(X) ^ (Y)> (power)
Raises X to the Y power (it's basically an exponent block).
<(X) // (Y)> (truncated division)
Truncated (or integer) division. Returns how many times Y fits in to X, without Y times that number being greater than X.
(This is the same as floor(X / Y).)

(pi) (π)
Returns the irational number equivalent to pi (π), this is around 3.141529...
(e) (Euler's number)
Returns the irational number equivalent to e, this is around 2.718281...
